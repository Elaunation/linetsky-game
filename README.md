# linetsky-game

Player Setup
Support 1-6 players or groups on separate browser but the same game session
Enter player or group name
Timer System
Default: 7 minutes per game session
Adjustable in 1 minute increments. Minimum 5 minutes and max 10 minutes
Game Mechanics
One group comes up with an Input sentence (e.g., "Cat Ate Rice")
Each game session has two rounds.
First round, Pair each letter of the sentence after each alphabet sequence:
1st letter + A (A + C = AC)
2nd letter + B (B + A = BA)
3rd letter + C (C + T = CT)
Second round, Pair each letter of the sentence with with alphabet sequence:
1st letter + A (C + A = CA)
2nd letter + B (A + B = AB)
3rd letter + C (T + C = TC)
Gameplay
Player or group must guess a single famous people with each given initials
Name validation system utilizing wikipedia (celebrities, historical figures, fictional characters)
Once the timer is up, the game will check the names provided and award 1 point per valid name
Once the first round is completed the teams will compare scores. The second session will then start when each player or group presses ready.
At the end of the second round, add the first round and second round score together. The player or group with the highest total score between both rounds wins. 
Scoring & Win Condition
Track score for each player or group across the round. Reset when a new game session starts.
Additional Features
Win screen, include an face icon of one of the famous names used
Game reset/new session
Score display
Include an option where the player or group can check if the names they have guessed so far are valid or not mid round. Each round gets one check.
